## Features

- Add and remove candles with buttons  
- Animated flickering flames  
- Responsive candle row layout  
- Built with React

## Installation

1. Clone the repository:  
  git clone https://github.com/your-username/birthday-candle-app.git
  Install dependencies:

2. Install dependencies:
  cd birthday-candle-app
  npm install

3. Run the app:
  npm start

Usage
Click Add Candle to add a candle

Click Remove Candle to remove a candle

Flames flicker with randomized animation delays
